Control Flow

Control flow structures in C, such as if, else, and switch, allow you to make decisions in your program based on conditions.

if-else statement: The if-else statement in C is a conditional statement that allows a program to execute different blocks of code based on a specified condition. The syntax for the if-else statement is as follows:

Syntax:

if (condition)
{ // code to be executed if the condition is true }
else
{ // code to be executed if the condition is false }

switch statement: The switch statement in C is a control statement that allows a variable to be tested for equality against a list of values. Here's the basic syntax of a switch statement:

Syntax:

switch (expression)
{ case 1:
// code to be executed if expression is equal to constant1
break;

case 2:
// code to be executed if expression is equal to constant2
break;

// more cases can be added as needed

default:
// code to be executed if expression doesn't match any of the constants
}

Loops

Loops, including for, while, and do-while, enable you to repeat a block of code multiple times based on a specified condition.

Loops are fundamental for controlling the flow of a program and are commonly used for tasks like iterating through arrays, processing data, and implementing repetitive tasks.
Each type of loop has its own use cases, and the choice of loop depends on the specific requirements of the task at hand.

for loop: The for loop is used when the number of iterations is known in advance. It consists of three parts: initialization, condition, and update. Initialization is executed once at the beginning, the condition is checked before each iteration, and the update is executed after each iteration.

while loop: The while loop is used when the number of iterations is not known in advance. It continues to execute the loop body as long as the specified condition is true. The condition is checked before entering the loop, and if it is false initially, the loop body may not execute at all.

do-while loop: The do-while loop is similar to the while loop but guarantees that the loop body is executed at least once. It first executes the loop body and then checks the condition. If the condition is true, it continues to execute the loop; otherwise, it exits

Arrays and Pointers

Arrays and pointers are crucial concepts for handling collections of data in C.

1. Arrays:
- An array is a collection of elements of the same data type stored in contiguous memory locations. - Elements in an array are accessed using an index, starting from 0 for the first element. - Arrays provide a convenient way to store and manipulate a fixed-size sequence of values.

2. Pointers:
- A pointer is a variable that stores the memory address of another variable. - Pointers are used to work with dynamic memory allocation, arrays, and to create more flexible functions. - The dereference operator (*) is used to access the value at the memory address stored in a pointer.

3. Array Pointers:
- Arrays and pointers are closely related in C. The name of an array can be considered as a pointer to the first element of the array. - Pointer arithmetic allows you to navigate through array elements using pointer operations.

4. Pointer Arithmetic:
- Pointer arithmetic involves adding or subtracting an integer from a pointer, which adjusts the pointer's address based on the size of the data type. - For example, if ptr is a pointer to an integer, ptr + 1 points to the next integer in memory.

5. Dynamic Memory Allocation:
- Pointers are commonly used with functions like malloc, calloc, and realloc to dynamically allocate memory at runtime. - Dynamic memory allocation allows you to create data structures with a variable size.

6. Null Pointers:
- A null pointer is a pointer that does not point to any memory location. It is represented by the value NULL in C. - Null pointers are used to indicate that a pointer does not currently point to a valid object.

7. Pointer to Functions:
- Pointers can also point to functions, allowing you to create more flexible and generic code. - This is particularly useful for implementing callback functions and function pointers.

Structures and Unions

Structures and unions enable you to create complex data types by grouping different variables together.

In this the structure and unions are same only the keywords are different


A structure is a composite data type that groups together variables of different data types under a single name. Each variable within a structure is called a member or field.
Structures allow you to create complex data structures to represent real-world entities with multiple attributes.

Declaration of Structures:
To define a structure, you use the struct keyword followed by the structure name and a list of members enclosed in curly braces.

Unions:
A union is a special data type that allows you to store different data types in the same memory location.
Unlike structures, where all members have their own memory space, in a union, all members share the same memory.
Unions are useful when you need to represent a value that can be of different types at different times.

Declaration of Unions:
Similar to structures, you use the union keyword to define a union, followed by the union name and a list of members enclosed in curly braces.

   
    
    #include <string-.h >
      
      // create struct with person1 variable

      int main() {

        person1. salary = 2500;
      
        // print struct variables
        printf("Name: %s\n", person1.name);
        printf("Citizenship No.: %d\n", person1.citNo);

Unions:

      #include <stdio.h >
        union Job {
           float salary;
           int workerNo;
        } j;
        
        int main() {
           j.salary = 12.3;
        
           // when j.workerNo is assigned a value,

           printf("Salary = %.1f\n", j.salary);

        }

File Handling

File handling in C involves operations like reading from and writing to files, which is essential for data persistence.

    
    
fileHandling:

    #include <stdio.h >
    
      int main() {
          // File pointer
          FILE *filePointer;
      
          // Open file for writing
          filePointer = fopen("example.txt", "w");
      
          // Check if the file is opened successfully
          if (filePointer == NULL) {
              printf("Error opening the file.\n");
              return 1; // Exit with an error
          }
      
          // Write content to the file
          fprintf(filePointer, "Hello, File Handling in C!");
      
          // Close the file
          fclose(filePointer);
      
          // Open file for reading
          filePointer = fopen("example.txt", "r");
      
          // Check if the file is opened successfully
          if (filePointer == NULL) {
              printf("Error opening the file.\n");
              return 1; // Exit with an error
          }
      
          // Read and print content from the file
          char content[100];
          fscanf(filePointer, "%[^\n]", content);
          printf("File Content: %s\n", content);
      
          // Close the file
          fclose(filePointer);
      
          return 0; // Exit successfully
      }

C Programming Interview Questions

  • 1. What is the difference between malloc and calloc?
    • Ans: malloc and calloc are both functions in the C programming language that are used for dynamic memory allocation, but they have some key differences.

  • 2.Explain the concepts of pass by value and pass by reference in C.
    • Ans: In C, when you pass arguments to a function, the method by which these arguments are passed can be categorized into two main concepts: pass by value and pass by reference.
  • 3.How are arrays and pointers related in C?
    • Ans: In C, arrays and pointers are closely related, and understanding their relationship is fundamental to working with C programming. Here are some key points to understand the relationship between arrays and pointers

  • 4.What is the purpose of the const keyword in C?
    • Ans: In the C programming language, the const keyword is used to declare constants and to indicate that a variable's value should not be modified after it has been initialized. It can be applied to variables, pointers, and function parameters.

  • 5.How does the sizeof operator work in C?
    • Ans: The volatile keyword is used to indicate that a variable may be changed at any time outside the current code, preventing the compiler from optimizing it.
    In C, the sizeof operator is used to determine the size, in bytes, of a variable or a data type.

  • 6. What is the purpose of the volatile keyword in C?
    • Ans: In C, the volatile keyword is used to indicate that a variable's value might change at any time without any action being taken by the code.
    It tells the compiler not to optimize or cache the value of the variable because it can be modified externally.

  • 7.Explain the difference between ++i and i++ in C.
    • Ans: In C, both ++i and i++ are increment operators. ++i is the pre-increment operator, which increments the value of i and then uses the updated value in the expression.
    i++ is the post-increment operator, which uses the original value of i in the expression and then increments it.

  • 8.What are the differences between structures and unions in C?
    • Ans: Structures (struct):
    Members have their individual memory spaces. Each member holds different information. Size is the sum of member sizes. All members can be accessed independently. Used for storing multiple types of data simultaneously.
    Unions (union):
    Members share the same memory space. Only one member is active at a time. Size is determined by the largest member. Modifying one member affects others. Ideal when only one type of data needs storage or access at any given time.

  • 9.How do you use enum in C to define constants?
    • Ans: In C, you can use enum (enumeration) to define a set of named integer constants, providing a way to create symbolic names for integral values.

  • 10.What is a pointer to a function in C? Provide an example.
    • Ans: In C, a pointer to a function is a variable that stores the memory address of a function. It allows you to call a function indirectly through the pointer, providing flexibility in function invocation.
    The syntax for declaring a function pointer involves specifying the return type and parameter types of the function it points to

  • 11.Explain the purpose of the #include directive in C.
    • Ans: The #include directive aids in creating modular and maintainable code by allowing developers to organize their code into separate files and reuse components across different parts of a program.

  • 12.What is the role of the static keyword in C?
    • Ans: the static keyword in C can be used to control the scope and lifetime of variables and functions, both within functions and at the global level.

  • 13.How does memory management work in C? Explain the use of free() function.
    • Ans: Memory management in C involves the allocation and deallocation of memory during program execution. C provides several functions for dynamic memory allocation, and the free() function is used for deallocating memory that was previously allocated by functions like malloc(), calloc(), or realloc().

  • 14.What is a preprocessor in C? Explain some commonly used preprocessor directives.
    • Ans: In C, the preprocessor is a tool that performs text manipulations on the source code before it is compiled.
    It handles directives, which are commands beginning with the # symbol.

  • 15.How do you implement file I/O operations in C?
    • Ans: Input/output (I/O) operations in C involve reading data from standard input (keyboard) or other sources, and writing data to standard output (console) or other destinations like files.
